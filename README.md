# React Movie Search App

A responsive movie search web application built using React.js that allows users to search and explore movies in real time using an external API.

## Features
- Search movies dynamically based on user input
- Display movie details such as title, poster, and rating
- Reusable and modular component-based architecture
- Responsive UI for different screen sizes
- Handles loading and error states efficiently

## Tech Stack
- React.js
- JavaScript (ES6+)
- CSS
- REST API (OMDb)

## Description
This project demonstrates core React concepts including component-based design, state management using hooks (useState, useEffect), API integration, and conditional rendering. It is designed to provide a smooth and interactive user experience while maintaining clean and maintainable code structure.
